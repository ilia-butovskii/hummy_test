# Backend

Run local mongodb

Go to the backend folder

```bash
cd backend
```

*Edit src/config.ts file to change cache ttl and mongo uri*

Install dependencies

```bash
npm install
```



Build

```bash
npm run build
```

Test

```bash
npm run test:backend
```

Run server

```bash
npm run start:prod
```

# Frontend

Go to the frontend folder

```bash
cd frontend
```

Install dependencies

```bash
npm install
```

Run server

```bash
npm run dev
```
